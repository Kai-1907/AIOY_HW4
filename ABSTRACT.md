### 📝 專案摘要 (ABSTRACT)
專題名稱：整合 CNN 影像分類與多模態 LLM 之智能食物營養分析 Agent

本專題旨在實作一個從電腦視覺辨識延伸至自然語言生成的完整 AI 應用。實作核心採用卷積神經網絡（CNN）架構中的 MobileNetV2 模型，針對使用者上傳之食物影像進行特徵提取與即時分類。針對深度學習模型在 Streamlit 雲端部署時常遇到的 GPU 資源限制問題，本專案採取了環境優化策略，透過 tensorflow-cpu 套件與模型緩存技術，確保在無 GPU 加速的環境下仍能達成高效能推論。

專案的延伸創新在於導入 AIGC Agent 架構。系統在取得深度學習模型的預測標籤後，自動將數據傳輸至 Google Gemini Pro API。透過 Prompt Engineering 的設定，AI Agent 能精確扮演「專業營養師」與「美食評論家」之雙重角色，將枯燥的分類結果轉化為包含營養價值分析、熱量預估及社群媒體行銷文案的加值內容。本實作不僅展示了深度學習模型的部署技巧，更體現了 AIGC 如何賦予傳統影像辨識系統更高的互動性與應用價值。
